Що таке HTML?

HTML (Hypertext Markup Language) - це мова розмітки, яка використовується для створення веб-сторінок.

Основні приклади HTML

Ось основний приклад структури HTML-документа:

            <!DOCTYPE html>
            <html>
            <head>
                <title>Title of the document</title>
            </head>
            <body>
                Content of the document......
            </body>
            </html>

HTML Елементи та Атрибути

HTML складається з елементів, які використовуються для розмітки контенту на веб-сторінках.

  • Елемент параграфу: <p>
  • Елемент посилання: <a>
  • Елемент поділу: <div>
